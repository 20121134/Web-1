Ayuda :
  • No introduzca operadores lógicos (AND, OR o AND NOT) entre las palabras. Separe las palabras con espacio. Utilice la opción "Todas las palabras" (AND) para relacionar las palabras (restringe la búsqueda). La opción "Cualquier palabra" (OR) para sumar las palabras (amplia la búsqueda).
  • Use el símbolo de truncamiento $ para buscar palabras con la misma raíz. Ejemplo: educ$ recupera educación, education, educação, etc.
  • Para buscar por otros campos o para especificar el campo de búsqueda, use el Formulario Básico.
